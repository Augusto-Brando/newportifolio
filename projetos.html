Meus Projetos

Apple:

Apple Inc. é uma empresa multinacional norte-americana que tem o objetivo de projetar e comercializar produtos eletrônicos de consumo, software de computador e computadores pessoais. Neste projeto utilizei as linguagens: PHP para o back-end, HTML5 e CSS3 para front-end e JavaScript para interações.

Amazon:

Amazon.com, Inc. é uma empresa multinacional de tecnologia norte-americana com sede em Seattle, Washington. A companhia se concentra no e-commerce, computação em nuvem, streaming e inteligência artificial. Neste projeto auxiliei com meus conhecimentos em PHP para o back-end

Tesla:

Tesla, Inc. é uma empresa automotiva e de armazenamento de energia norte americana que desenvolve, produz e vende automóveis elétricos de alto desempenho, componentes para motores e transmissões para veículos. Neste projeto auxiliei com meus conhecimentos em JavaScript para desenvolver os sites.

Verizon:

A Verizon Communications Inc. é uma sociedade estadunidense especializada em telecomunicações. Tem mais de 200 milhões de clientes espalhados por todo o mundo e é parceira da Hyette Communications. Neste projeto auxiliei no back-end utilizando PHP e SQL para banco de dados

Tik Tok:

TikTok, também conhecido como Douyin e anteriormente Musical.ly na China, é um aplicativo de mídia para criar e compartilhar vídeos curtos. Neste projeto em específico, auxiliei no FullStack, desenvolvendo interfaces para os sistemas com JavaScript e também PHP e SQL para tratamento de dados

Google:

Google é uma empresa de tecnologia americana com foco em inteligência artificial, publicidade online, tecnologia de mecanismo de busca, computação em nuvem, software de computador. Neste projeto auxiliei no desenvolvimento de interfaces utilizando HTML5 CSS3 e JavaScrip.
