Minha formação

Atualmente estou cursando Sistemas de Informação pela Faculdade Metodista Granbery em Juiz de Fora, Minas Gerais.

Tenho o prazer de compartilhar minha formação em três cursos fundamentais: HTML e CSS, JavaScript e React. O curso de HTML e CSS me proporcionou uma base sólida para construir páginas web dinâmicas e visualmente atraentes. Com essas linguagens, aprendi a estruturar o conteúdo e estilizar os elementos, garantindo uma experiência de usuário agradável.

Em seguida, mergulhei no mundo do JavaScript, uma linguagem de programação essencial para tornar meus projetos interativos e funcionais. Através do JavaScript, fui capaz de adicionar recursos dinâmicos, como validação de formulários, manipulação de eventos e animações, elevando a interatividade das minhas páginas web.

Além disso, aprofundei meus conhecimentos com o curso de React, uma biblioteca JavaScript amplamente utilizada para criar interfaces de usuário reativas e eficientes. Com o React, pude desenvolver aplicativos web modernos e responsivos, com componentes reutilizáveis e gerenciamento de estado eficiente.

Esses cursos foram de extrema importância para minha formação na área de desenvolvimento web. Eles me permitiram adquirir habilidades técnicas sólidas, possibilitando a criação de websites e aplicações web altamente funcionais e visualmente atraentes. Estou entusiasmado em aplicar esses conhecimentos em projetos reais, colaborando com equipes e trazendo soluções inovadoras para o mundo da tecnologia.

Com essas bases sólidas em HTML e CSS, JavaScript e React, estou preparado para enfrentar os desafios do desenvolvimento web e contribuir para projetos inovadores. Estou sempre buscando aprender mais e me manter atualizado com as últimas tendências e tecnologias na área. Mal posso esperar para utilizar essas habilidades em novos projetos e ajudar a construir o futuro da web."
